Vacía el formulario Crea botón Crea un campo de ingreso oculto Crea una caja de texto enmascarada Tipo email Permite ingresar un cuadro de texto para cargar un email que es validado(con HTML5) Tipo dirección url Permite ingresar direcciones WEB absolutas, retorna error si el formato no es válido. Tipo teléfono Permite ingresar números de teléfono, no requiere sintaxis particular , es una indicación para el navegador de acuerdo al dispositivo en el cual la aplicación es ejecutada. Tipo número Permite ingresar entradas numéricas. En el ejemplo, si se declara un valor 5 para el atributo step en un campo que tiene valor mínimo de 0 y máximo de 10, el navegador no permitirá especificar valores entre0 y 5 y entre 5 y 10. Tipo rango Es un nuevo control que permite al usuario elegir un valor a partir de un rango. Tipo fecha Es un nuevo control que muestra un calendario cuando el usuario hace clic sobre el campo. El valor generado y esperado tiene la sintaxis año-mes-día. Tipo semana Es un nuevo control que permite seleccionar una semana en el año. Su sintaxis es año-semana (2016-w50) Tipo mes Es un nuevo control que permite seleccionar un mes en el año. Su sintaxis es año-mes. Tipo hora Es un nuevo control que permite seleccionar una hora. Su sintaxis es hora-minutos-segundos. Tipo fecha/hora Sirve para ingresar la fecha y hora completa, incluyendo la franja horaria.
